# react-native-xlog-jsi

It allows you to easily use MMKV inside your React Native applications.

## Installation

```sh
npm install react-native-xlog-jsi
```

## Usage

```js
import { multiply } from "react-native-xlog-jsi";

// ...

const result = await multiply(3, 7);
```
